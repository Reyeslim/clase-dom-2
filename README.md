# Clase 04 Live Coding 2 - Ejercicios DOM en 8 Pasos

Ejercicios progresivos para aprender manipulación del DOM desde cero.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Flexbox para layouts. Responsive
- **JavaScript ES6**:
  - DOM manipulation
  - Event listeners
  - Array methods (map, filter)
  - Arrow functions
  - Template literals

**Puntos clave a enfatizar**:

1. La diferencia entre textContent e innerHTML
2. Cuándo usar createElement vs innerHTML
3. Event listeners en elementos dinámicos
4. Importancia de limpiar contenedores antes de renderizar
